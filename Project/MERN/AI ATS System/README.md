# AI ATS System

A MERN Stack Applicant Tracking System that helps recruiters manage job postings and candidates track applications efficiently.

## Features

### Recruiters
- Create and Manage Jobs
- View Applications
- Update Candidate Status
- Dashboard Overview

### Candidates
- Register and Login
- Browse Available Jobs
- Apply to Jobs
- Track Application Status

## Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js
- JWT Authentication

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create a `.env` file in the backend directory:

```env
PORT=8000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
```
